This library is currently an (a)sync wrapper for
https://royaleapi.com and the official Clash Royale API with
great functionality.

Installation
============

Development build
.. code-block:: python

    pip install git+https://github.com/cgrok/clashroyale

Stable build
.. code-block:: python

    pip install clashroyale
